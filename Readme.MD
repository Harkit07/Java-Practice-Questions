# Java DSA Practice

A collection of Java programs implementing core data structures and algorithms, built for practice and interview preparation.

## 📂 Topics Covered

### Data Structures
- **Arrays & Matrix** — `Array.java`, `Matrix.java`
- **Linked List** — `LinkedList.java`, `LinkedList2.java`, `DoubleLinkedList.java`
- **Stack** — `StackB.java`
- **Queue & Deque** — `QueueB.java`, `DequeB.java`
- **Trees** — `Tree.java`, `BST.java` (Binary Search Tree)
- **Tries** — `Tries.java`
- **Heap** — `HeapDSA.java`
- **HashMap** — `HashMapCode.java` (custom implementation)
- **Priority Queue** — `PriorityQ.java`
- **Map & Set** — `MapandSet.java`
- **ArrayList** — `ArrayListJava.java`

### Algorithms
- **Graph Algorithms** — `GraphDSA.java`, `GraphAlgorithm.java`, `GraphAlgorithm2.java`, `GraphSupplement.java`
- **Dynamic Programming** — `DP.java`
- **Greedy Algorithms** — `GreedyAlgorithm.java`
- **Backtracking** — `Backtracking.java`
- **Recursion** — `Recursion.java`

### Other
- **Strings** — `Strings.java`
- **Pattern Printing** — `Pattern.java`

## 🚀 Getting Started

### Prerequisites
- [Java JDK](https://www.oracle.com/java/technologies/downloads/) installed (JDK 8 or above recommended)

### Running a file
```bash
# Compile
javac FileName.java

# Run
java FileName
```

Example:
```bash
javac BST.java
java BST
```

## 📁 Project Structure

Each `.java` file is self-contained and can be compiled/run independently. Most files include helper classes/inner classes (e.g., `Node`, `Edge`, `Pair`) to support the main data structure or algorithm implementation.
